标签用于对 Agent 进行分类和标记。每个 Agent 可以拥有多个标签，标签会显示在管理后台和公开状态页中。

## 语法格式

```
名称<颜色,图标>
```

- **名称**（必填）：标签显示的文本。
- **颜色**（可选）：从预设色板中选择，默认为 `blue`。
- **图标**（可选）：从预设图标中选择，默认为 `tag`。

颜色和图标均为可选，以下写法都是合法的：

| 输入                  | 效果                               |
| --------------------- | ---------------------------------- |
| `Production`          | 文本 "Production"，蓝色，tag 图标  |
| `CN2<green>`          | 文本 "CN2"，绿色，tag 图标         |
| `DB<violet,database>` | 文本 "DB"，紫罗兰色，database 图标 |

批量添加时使用**英文分号**分隔多个标签，例如：

```
CN2<blue,route>; NAT<orange>; Primary<green,star>
```
