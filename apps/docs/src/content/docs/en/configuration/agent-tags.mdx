Tags are used to categorize and label agents. Each agent can have multiple tags, which are displayed in both the admin panel and the public status page.

## Syntax

```
name<color,icon>
```

- **name** (required): The text displayed on the tag.
- **color** (optional): A color from the preset palette. Defaults to `blue`.
- **icon** (optional): An icon from the preset icon set. Defaults to `tag`.

Both color and icon are optional. All of the following are valid:

| Input                 | Result                            |
| --------------------- | --------------------------------- |
| `Production`          | Text "Production", blue, tag icon |
| `CN2<green>`          | Text "CN2", green, tag icon       |
| `DB<violet,database>` | Text "DB", violet, database icon  |

To add multiple tags at once, separate them with **semicolons**:

```
CN2<blue,route>; NAT<orange>; Primary<green,star>
```
